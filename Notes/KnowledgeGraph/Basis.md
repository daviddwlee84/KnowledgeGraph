## Knowledge Graph

Definition: Knowledge graph is a technical approach that use **graph model/structure** to *represent knowledge* and *modeling the relation between everything*.

* Node
  * Entity
    * e.g. a person, a book
  * Abstract concept
    * e.g. AI, knowledge graph
* Edge
  * Property
    * e.g. person name, book name
  * Relation
    * e.g. friend, couple
